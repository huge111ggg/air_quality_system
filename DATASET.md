# 数据集说明

## 数据来源

本项目使用课程设计演示用模拟城市空气质量数据，由 `generate_dataset.py` 按固定随机种子生成。数据不对应真实城市和真实监测站点，仅用于机器学习流程演示。

## 数据规模

- 样本数量：220 条
- 字段数量：13 个
- 任务类型：多分类预测
- 标签字段：`quality_level`

## 字段说明

| 字段名 | 含义 | 类型 |
| --- | --- | --- |
| temperature | 温度 | 数值 |
| humidity | 湿度 | 数值 |
| wind_speed | 风速 | 数值 |
| rainfall | 降雨量 | 数值 |
| traffic_index | 交通拥堵指数 | 数值 |
| industrial_index | 工业排放指数 | 数值 |
| pm25 | PM2.5浓度 | 数值 |
| pm10 | PM10浓度 | 数值 |
| no2 | 二氧化氮浓度 | 数值 |
| so2 | 二氧化硫浓度 | 数值 |
| ozone | 臭氧浓度 | 数值 |
| aqi | 空气质量指数 | 数值 |
| quality_level | 空气质量等级：优、良、轻度污染、中度及以上污染 | 标签 |

## 预处理方法

- 缺失值处理：数值特征使用中位数填充。
- 异常值处理：对温度、湿度、风速、降雨、污染物浓度等字段进行合理范围裁剪。
- 特征标准化：使用 `StandardScaler` 对数值特征进行标准化。
- 训练测试划分：按 75%/25% 划分，并使用分层抽样保持类别比例。
